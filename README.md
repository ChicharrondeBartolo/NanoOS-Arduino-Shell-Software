# NanoOS-Arduino-Shell-Software
Un SO adaptado para Arduino UNO-R3 con consola de comandos. Es necesario usar la consola del IDE de Arduino con las opciones "new line" y "9600 baud"

----------------------------------------------------------------------------------------------NanoOS/V1.0--------------------------------------------------------------------------------------------------------------

El sistema más básico. Control del LED TX, uso del rendimiento y capacidad de reiniciar el sistema.

----------------------------------------------------------------------------------------------NanoOS/V2.0--------------------------------------------------------------------------------------------------------------

Ahora con capacidad de iniciar sesión. Puedes cambiar de contraseña y de nombre de usuario en el código.

----------------------------------------------------------------------------------------------NanoOS/V3.0--------------------------------------------------------------------------------------------------------------

Capacidades más avanzadas y con más comandos. Se ha quitado la función de iniciar sesión. Capacidad de iniciar y parar procesos. Puede mostrar lineas de procesos o la RAM libre.

----------------------------------------------------------------------------------------------NanoOS/V4.0--------------------------------------------------------------------------------------------------------------

Igual que la versión V3.0 pero con la capacidad de jugar tres en raya contra la máquina.
